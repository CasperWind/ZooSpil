# ZooSpil.

### spillet.
Dette zoo spil skal være text baseret med kanpper og text. Man skal tjæne penge pr sekunt. Og købe dyr til ens zoo. Man starter med 10.000 zoomers (Penge i spillet)
Og får mulighed for at købe et af 3 dyr (Elefant, abe eller tiger), også begynder spillet.
